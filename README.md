# DSCI 345 Probability & Statistics for Data Science

This repo contains my homework assignments for DSCI 345, a course covering the theoretical foundations of probability and statistics for data science.

---

## 📚 Course Overview

Topics covered include:

- Probability, expectation, conditional probability, and random variables
- Simulating random data from real-world models
- Model fitting: method of moments, minimum variance estimators, CLT
- Hypothesis testing, likelihood, and p-values
- Measuring power, false positives, and false discovery rates
- Principal component analysis (PCA) on numerical and text data
- Overfitting, regularization, and cross-validation
